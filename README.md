### How to use this repository:
1.  **Clone the Repo:**
    ```bash
    git clone https://github.com/Lokesh19-PP/Hack-Series---Session.git
    ```
2.  **Install Dependencies:**
    ```bash
    npm install
    ```
3.  **Run Development Server:**
    ```bash
    npm run dev
    ```
